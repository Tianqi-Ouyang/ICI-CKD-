Pipeline overview

Stage Inputs Output
1. Demographics & cohort ICI starts registry + RPDR Dem Race, Ethnic_Group, sex, last-known-alive date
2. Pre-baseline labs (within 180 days of ICI) RPDR Lab pre_CRE_180days, pre_HGB_180days, pre_ALB_180days
3. Baseline eGFR Pre-baseline CRE + age + sex eGFR_cre via CKD-EPI 2021 race-free formula
4. Comorbidities & medications on/before ICI RPDR Dia + Med dm, htn, cad, cirrhosis, esrd_kt, ppi, ace_arb, diu, steroids, smoking
5. Post-ICI CKD outcomes Serial CRE after baseline + CKD-EPI ckd_incidence, ckd_progression, eskd_defination_1 (each with 90-day persistence rule)
6. ESRD / KT after ICI RPDR Dia (post-ICI date) esrd_kt_after_ici, esrd_kt_after_ici_date
7. Cohort exclusions ICI date < 2024-03-31; baseline CRE exists; no pre-existing ESRD/KT; baseline eGFR ≥ 10; no RPDR death-date errors
8. ICI class & companion chemo (±30 days of ICI) RPDR Med ICI_Class (PD1/PDL1/CTLA4/Combination), Bevacizumab, Cisplatin, Carboplatin, Pemetrexed, Gemcitabine, Cetuximab, Trastuzumab, VEGF_TKI, nephrotoxic_chemo
9. AKI from serial CRE RPDR Lab (post-ICI CRE) aki, aki_date, aki_days using KDIGO-style rules (CRE > 1.5× median-90d OR +0.3 over lowest-2d)
10. Composite outcomes & competing-risk status All outcomes + death eskd_composite, ckd_composite, status_ckd_cmp (Censored / CKD / Death), time_to_ckd_cmp
11. Yearly eGFR & CKD stage (1–10 years) Yearly median CRE per patient eGFR_CRE_1year…10year, ckd_stage_1year…10year

Key method decisions

  • eGFR formula — CKD-EPI 2021 race-free creatinine-based equation for all eGFR calculations (baseline, yearly, pre/post-event).
  • Outcome definitions — CKD events require the defining condition to hold for ≥ 90 days (persistence rule) before being counted, to exclude transient dips.
  • AKI — post-ICI creatinine compared to both a 90-day pre-window median (KDIGO 1.5× criterion) and a 2-day pre-window minimum (+0.3 mg/dL criterion).
  • Competing risk — death is modeled as a competing event to CKD composite. Patients who die before any CKD event contribute to the Death stratum of status_ckd_cmp.
  • Pre-index windows — all baseline covariates (medications, diagnoses, labs) are restricted to dates on or before ICI_Dose_Date to prevent temporal leakage.

ckd_epi_gfr <- function(creat, male, age) {
  k <- ifelse(male == 1, 0.9, 0.7)
  a <- ifelse(male == 1, -0.302, -0.241)
  m <- ifelse(male == 1, 1, 1.012)
  gfr <- 142 * min(c(creat/k, 1))^a *
               max(c(creat/k, 1))^-1.2 *
               0.9938^age * m
  return(gfr)
}

## 3.7 make_ckd_outcome_date — sustained-episode outcomes

Identifies the first episode where a condition (e.g., `eGFR < 60`) is sustained
for ≥ `days_threshold` days and returns both the binary flag and first-event date.

```{r}
#| label: fn-make-ckd-outcome-date
make_ckd_outcome_date <- function(data, id_var, date_var, condition_expr,
                                  days_threshold = 90, outcome_name = "default") {
  id_var      <- enquo(id_var)
  date_var    <- enquo(date_var)
  cond_q      <- enquo(condition_expr)
  outcome_sym <- ensym(outcome_name)
  outcome_date_name <- paste0(outcome_name, "_date")

  working <- data %>%
    arrange(!!id_var, !!date_var) %>%
    group_by(!!id_var) %>%
    mutate(
      .cond   = if_else(!!cond_q, 1L, 0L),
      .change = .cond != lag(.cond, default = first(.cond)),
      .run    = cumsum(.change)
    )

  episodes <- working %>%
    group_by(!!id_var, .run) %>%
    summarise(.code = first(.cond),
              .start = first(!!date_var),
              .last  = last(!!date_var),
              .groups = "drop") %>%
    group_by(!!id_var) %>%
    mutate(.end  = lead(.start),
           .end  = if_else(is.na(.end), .last, .end),
           .days = as.integer(.end - .start)) %>%
    ungroup()

  episodes %>%
    group_by(!!id_var) %>%
    summarise(
      has_event  = any(.code == 1L & .days >= days_threshold),
      event_date = if (any(.code == 1L & .days >= days_threshold))
                      min(.start[.code == 1L & .days >= days_threshold])
                   else as.Date(NA),
      .groups = "drop"
    ) %>%
    mutate(!!outcome_sym := as.integer(has_event)) %>%
    rename(!!outcome_date_name := event_date) %>%
    select(-has_event)
}
```

## 3.8 create_aki — KDIGO-style AKI from serial CRE

```{r}
#| label: fn-create-aki
create_aki <- function(data,
                       index_date = ICI_Dose_Date,
                       Group_Id   = EMPI,
                       median_window_days = 90,
                       lowest_window_days = 2) {
  index_date <- enquo(index_date)
  Group_Id   <- enquo(Group_Id)

  df <- data %>%
    filter(Group_Id == "CRE", grepl("^\\d+\\.\\d+$", Result)) %>%
    filter(!is.na(Result)) %>%
    mutate(Result = as.numeric(Result)) %>%
    rename(cre_date = Seq_Date_Time, cre = Result) %>%
    select(EMPI, cre, cre_date, !!index_date) %>%
    arrange(!!Group_Id, cre_date)

  df %>%
    filter(cre_date > !!index_date) %>%
    group_by(!!Group_Id) %>%
    mutate(
      median_cre_90days = sapply(seq_along(cre), function(i) {
        vals <- cre[cre_date >= cre_date[i] - median_window_days &
                    cre_date <= cre_date[i] - 1]
        if (length(vals) == 0) NA_real_ else median(vals, na.rm = TRUE)
      }),
      lowest_cre_2days = sapply(seq_along(cre), function(i) {
        vals <- cre[cre_date >= cre_date[i] - lowest_window_days &
                    cre_date <= cre_date[i] - 1]
        if (length(vals) == 0) NA_real_ else min(vals, na.rm = TRUE)
      }),
      aki_flag = (!is.na(median_cre_90days) & cre > 1.5 * median_cre_90days) |
                 (!is.na(lowest_cre_2days)  & cre - lowest_cre_2days >= 0.3)
    ) %>%
    filter(aki_flag) %>%
    arrange(!!Group_Id, cre_date) %>%
    summarise(aki      = 1L,
              aki_date = min(cre_date), .groups = "drop")
}
```
