Project overview

This site publishes the Quarto pipeline for a retrospective cohort study examining chronic kidney disease (CKD) outcomes following immune checkpoint inhibitor (ICI) therapy.

The pipeline has two stages:

Stage File Role
Data management data_management.qmd Load raw ICI starts + RPDR files, derive demographics, comorbidities, labs, eGFR trajectories, CKD / ESKD / AKI outcomes, competing-risk status
Analysis analysis.qmd Baseline characteristics, yearly tables, CIF curves, logistic regression, Fine-Gray competing-risk models, cause-specific Cox models, overall survival by cancer type

Data privacy

Patient-level identifiers (EMPI, MRN, names, dates of birth) never appear in any rendered output. Rendered tables and plots contain only aggregate summaries well above the small-cell threshold; all cell counts shown are institutional-level cohort statistics.

Raw RPDR extracts and clinical Excel files remain local — only the rendered code and aggregate results are published here.

## Project overview

This site publishes the Quarto pipeline for a retrospective cohort study
examining **chronic kidney disease (CKD) outcomes following immune checkpoint
inhibitor (ICI) therapy**.

The pipeline has two stages:

| Stage | File | Role |
|---|---|---|
| **Data management** | [`data_management.qmd`](data_management.qmd) | Load raw ICI starts + RPDR files, derive demographics, comorbidities, labs, eGFR trajectories, CKD / ESKD / AKI outcomes, competing-risk status |
| **Analysis** | [`analysis.qmd`](analysis.qmd) | Baseline characteristics, yearly tables, CIF curves, logistic regression, Fine-Gray competing-risk models, cause-specific Cox models, overall survival by cancer type |

## Data privacy

Patient-level identifiers (`EMPI`, `MRN`, names, dates of birth) never appear

institutional-level cohort statistics.
